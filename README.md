# Financial Analytics Dashboard - Power BI

Interactive financial analytics dashboard built in Power BI analyzing FY2024 revenue, expenses, and profit data.

## Key Metrics
- Total Revenue: 28M PKR
- Total Expense: 9M PKR
- Net Profit: 20M PKR
- Profit Margin: 69.40%
- Budget Variance: 1M PKR (Favorable)

## Features
- Monthly revenue trend analysis
- Revenue vs Expense by Quarter comparison
- Expense breakdown by Sub-Category
- Department wise expense analysis
- Monthly financial summary table
- Interactive slicers (Quarter, Department, Category)
- DAX measures for KPI calculations
